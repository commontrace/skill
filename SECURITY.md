# Security Policy

## Reporting a Vulnerability

**Do not open a public issue for security vulnerabilities.**

Email **security@commontrace.org** with:

- A description of the vulnerability and its impact
- Steps to reproduce (proof-of-concept if available)
- The affected version, endpoint, or component

You will receive an acknowledgement within 72 hours. We aim to provide a remediation timeline within 7 days and will keep you updated through resolution. Please give us a reasonable window to ship a fix before any public disclosure.

There is no paid bug-bounty program yet. We credit reporters in the release notes for the fix unless you prefer to stay anonymous.

## Scope

CommonTrace spans four repositories — `server`, `mcp`, `frontend`, and `skill` — plus the live services at `*.commontrace.org`. Vulnerabilities in any of them are in scope. Report them all to the address above.
